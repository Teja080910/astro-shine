# Astro Vetri - App Features List

This document lists all the unique pages and features for the **Admin Panel**, **User App**, and **Astrologer App**, excluding the main website features. Duplicates across the apps have been consolidated in the main list.

---

## 1. Unified Unique Pages & Features List

Below is a consolidated list of all pages and features across the Admin Panel, User App, and Astrologer App with duplicates removed:

### Authentication & Onboarding
* **Login** (Admin, User, Astrologer)
* **Register** (User, Astrologer)
* **Onboarding / Walkthrough** (User, Astrologer)

### Profiles & Accounts
* **Edit Profile** (User, Astrologer)
* **Delete Account Function** (User, Astrologer)

### Core Astrology Features
* **Home Page / Dashboard**
* **Kundli API / User Kundli API**
* **Matchmaking API**
* **Daily Horoscope**
* **Panchang API**
* **Zodiac Sign API**
* **Live Astrologers List / Go Live**
* **Category-wise Astrologers List**
* **Mandir Pooja Section**

### Communication (Voice, Video, & Chat)
* **Chat, Voice & Video Call with Astrologers** (User, Astrologer)
* **Online/Offline Status Toggle** (Astrologer)
* **Chat, Voice & Video Call Logs/Data** (Admin)
* **All API Keys Management (Voice/Video/Chat)** (Admin)

### Financials & Transactions
* **Add Funds with Payment Gateway** (User)
* **Payment Gateway Settings** (Admin)
* **Withdraw Amount / Deposit & Withdrawal Section** (Admin, Astrologer)
* **Wallet Management** (User, Astrologer)
* **Order History** (User)
* **Transaction History / All Transaction Data** (Admin, User, Astrologer)
* **Astrologer Commission Setting** (Admin)
* **Chat/Call Commission Logs** (Astrologer)
* **Gifts System** (Send Gift, Receive Gift, Gifts Redeem)
* **Donation Page**

### Content & Admin Management
* **Admin Dashboard**
* **User Data Management** (Admin)
* **Astrologer Data Management** (Admin)
* **Astrologer Verification / Verification Section** (Admin, Astrologer)
* **News Section** (Admin)
* **Blog Section** (User, Astrologer)
* **Review Section** (Admin)
* **Report System** (Report Astrologer, Report User)
* **Notification for All / App-wide Notifications** (Admin, User, Astrologer)
* **Dynamic Page Link Update Settings** (Admin)
* **Website Content Editor** (Admin)
* **Shop Astro Product** (User)

### App Settings & Miscellaneous
* **Dark and Light Mode Toggle** (User, Astrologer)
* **Help and Support** (User, Astrologer)
* **Terms and Conditions** (User, Astrologer)
* **About App** (User, Astrologer)
* **Playstore Upload / Release Management** (User, Astrologer)
